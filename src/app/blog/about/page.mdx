# My Journey in Software Engineering

I am **Abdurahmon Mamadiyorov**, a Software Engineer from Tashkent, Uzbekistan. My journey into programming started in 2020, and since then, I've been continuously learning and evolving in the field of software engineering.

I specialize in **full-stack development** with expertise in **TypeScript**, **React**, **Node.js**, and **Golang**. I have experience building SaaS platforms, backend services, browser extensions, and developer tools used in production. I'm skilled in designing scalable systems, writing maintainable code, and shipping features from concept to deployment. I'm also an active community builder and mentor, contributing to the growth of the Go ecosystem and developer community in Uzbekistan.

## Professional Experience

### [Numeo](https://numeo.ai) (November 2025 - June 2026)

**Role:** Software Engineer

Owned features end-to-end across a freight logistics platform: Chrome extension, React admin dashboard, Node/TypeScript backend services, and Telegram Mini Apps.

- Built and maintained Chrome extension features covered by Playwright e2e tests, validated in real production use.
- Designed a gamified onboarding flow inside a Telegram Mini App from Figma, handling edge cases across user states beyond the happy path.
- Built a public Telegram factoring bot enabling carriers to get paid faster without leaving Telegram.
- Built a rich-text email template system allowing the ops team to self-serve transactional email edits.
- Built the full account onboarding flow — signup, activation, and profile setup.
- Integrated GA4, Amplitude, FMCSA, and GraphHopper for analytics, compliance, and routing.
- Maintained unit, integration, and e2e test coverage with Playwright across the stack; worked in a Rush monorepo with CI/CD on GitHub Actions, deployed on GCP.
