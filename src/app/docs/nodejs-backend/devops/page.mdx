# DevOps

DevOps amaliyotlari dasturlash va operatsiyalar o'rtasidagi bo'shliqni to'ldiradi, tezroq va ishonchliroq dasturiy ta'minot yetkazib berishni ta'minlaydi.

## Docker

Konteynerlashtirish asoslari:

```dockerfile
# Node.js ilovasi uchun Dockerfile
FROM node:18-alpine

WORKDIR /app

COPY package*.json ./
RUN npm install

COPY . .

EXPOSE 3000
CMD ["npm", "start"]
```

### Asosiy buyruqlar
- `docker build` - Image yaratish
- `docker run` - Konteyner ishga tushirish
- `docker-compose` - Ko'p konteynerli ilovalar

## Kubernetes

Konteyner orkestratsiyasi:

- **Pods** - Eng kichik deploy birligi
- **Services** - Tarmoq hizmatlar
- **Deployments** - Ilova versiyalarini boshqarish
- **ConfigMaps/Secrets** - Konfiguratsiya

```yaml
# Kubernetes deployment misoli
apiVersion: apps/v1
kind: Deployment
metadata:
  name: nodejs-app
spec:
  replicas: 3
  selector:
    matchLabels:
      app: nodejs
  template:
    metadata:
      labels:
        app: nodejs
    spec:
      containers:
      - name: nodejs
        image: nodejs-app:latest
        ports:
        - containerPort: 3000
```

## CI/CD

Uzluksiz integratsiya va deployment:

### GitHub Actions misoli
```yaml
name: CI/CD Pipeline

on:
  push:
    branches: [main]

jobs:
  build-and-deploy:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - uses: actions/setup-node@v3
        with:
          node-version: '18'
      - run: npm install
      - run: npm test
      - run: npm run build
```

### Asboblar
- **GitHub Actions** - GitHub integratsiyasi
- **Jenkins** - Ochiq kodli CI/CD
- **GitLab CI** - GitLab bilan integratsiya

## Monitoring

Kuzatuvchanlik va ogohlantirishlar:

- **Prometheus** - Metrikalar to'plash
- **Grafana** - Vizualizatsiya
- **ELK Stack** - Log boshqarish
- **Sentry** - Xato kuzatuvi

### Muhim metrikalar
- CPU va xotira ishlatilishi
- So'rov kechikishlari
- Xato darajasi
- Faol foydalanuvchilar soni
