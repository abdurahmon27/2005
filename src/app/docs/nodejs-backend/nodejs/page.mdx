# Node.js

Node.js - bu Chrome V8 JavaScript dvigateliga asoslangan server tomoni dasturlash muhiti. U tez, moslashuvchan va katta jamiyatga ega.

## Node.js asoslari

Node.js ning asosiy xususiyatlari:

- **Event-driven** - Hodisalarga asoslangan arxitektura
- **Non-blocking I/O** - Asinxron kiritish/chiqarish
- **npm** - Dunyodagi eng katta paket registri
- **Single-threaded** - Bitta ip, lekin samarali

```javascript
// Oddiy HTTP server
const http = require('http');

const server = http.createServer((req, res) => {
  res.writeHead(200, { 'Content-Type': 'text/plain' });
  res.end('Salom, dunyo!');
});

server.listen(3000, () => {
  console.log('Server 3000-portda ishlamoqda');
});
```

## Express.js

Express.js - Node.js uchun eng mashhur veb framework:

- **Routing** - URL yo'naltirish
- **Middleware** - So'rov/javob qayta ishlash
- **Template engines** - EJS, Pug, Handlebars
- **REST API** - Oson API yaratish

```javascript
const express = require('express');
const app = express();

app.get('/api/users', (req, res) => {
  res.json({ users: [] });
});

app.listen(3000);
```

## Asinxron dasturlash

Node.js da asinxron kod yozish usullari:

- **Callbacks** - Eski usul
- **Promises** - Zamonaviy yondashuv
- **async/await** - Eng qulay sintaksis

```javascript
// async/await misoli
async function getUsers() {
  try {
    const response = await fetch('/api/users');
    const data = await response.json();
    return data;
  } catch (error) {
    console.error('Xato:', error);
  }
}
```

## Testlash

Kod sifatini ta'minlash uchun testlash:

- **Jest** - Facebook tomonidan yaratilgan test framework
- **Mocha** - Moslashuvchan test framework
- **Supertest** - HTTP testlash
- **Unit va Integration testlar**

```javascript
// Jest test misoli
describe('User API', () => {
  test('should return users', async () => {
    const response = await request(app).get('/api/users');
    expect(response.status).toBe(200);
  });
});
```
