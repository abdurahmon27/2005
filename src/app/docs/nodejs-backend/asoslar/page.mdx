# Asoslar

Bu bo'lim backend dasturlashning asosini tashkil etuvchi muhim tushunchalarni qamrab oladi.

## HTTP asoslari

HTTP (HyperText Transfer Protocol) - bu veb-brauzerlar va serverlar o'rtasidagi aloqani ta'minlaydigan protokol.

- **So'rov metodlari** - GET, POST, PUT, DELETE, PATCH
- **Status kodlari** - 200 (OK), 404 (Not Found), 500 (Server Error)
- **Sarlavhalar** - Content-Type, Authorization, Cache-Control
- **So'rov va javob tanasi** - JSON, XML, form data

## REST API

REST (Representational State Transfer) - bu veb xizmatlarini yaratish uchun arxitektura uslubi.

- **Resurslar** - URL orqali manzillash
- **CRUD operatsiyalari** - Create, Read, Update, Delete
- **Statelessness** - Har bir so'rov mustaqil
- **JSON formati** - Ma'lumot almashish standarti

## Ma'lumotlar bazasi

Ma'lumotlarni saqlash va boshqarish uchun tizimlar:

### SQL ma'lumotlar bazalari
- PostgreSQL, MySQL, SQLite
- Strukturali ma'lumotlar uchun
- Jadvallar va munosabatlar

### NoSQL ma'lumotlar bazalari
- MongoDB, Redis, Cassandra
- Moslashuvchan sxemalar
- Katta hajmdagi ma'lumotlar uchun

## Autentifikatsiya

Foydalanuvchi identifikatsiyasi va xavfsizlik:

- **JWT (JSON Web Tokens)** - Token asosida autentifikatsiya
- **OAuth 2.0** - Uchinchi tomon autentifikatsiyasi
- **Session** - Server tomonida saqlash
- **Parol xeshlash** - bcrypt, Argon2

[Node.js](/docs/nodejs-backend/nodejs) bo'limiga o'ting va amaliy dasturlashni boshlang.
