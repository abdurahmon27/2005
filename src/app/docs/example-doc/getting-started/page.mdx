## Prerequisites

Before you begin, make sure you have:

- Node.js 18 or higher
- npm or yarn package manager
- A code editor (VS Code recommended)

## Installation

Install the package using npm:

```bash
npm install example-package
```

Or using yarn:

```bash
yarn add example-package
```

## Basic Setup

Create a configuration file `example.config.js`:

```javascript
module.exports = {
  // Basic configuration
  name: "my-project",
  version: "1.0.0",

  // Advanced options
  options: {
    debug: false,
    timeout: 5000,
    retries: 3
  }
};
```

## First Steps

1. Initialize your project:

```bash
npx example-cli init
```

2. Configure your settings in `example.config.js`

3. Run the development server:

```bash
npm run dev
```

## Project Structure

After initialization, your project will have this structure:

```
my-project/
├── src/
│   ├── index.js
│   └── utils/
├── example.config.js
├── package.json
└── README.md
```
